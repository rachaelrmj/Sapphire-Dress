# Sapphire Dress

A static fashion boutique website built with HTML and CSS to showcase a clean, visually appealing storefront for a women's clothing brand.

## Overview

Sapphire Dress is a front-end web design project that demonstrates the fundamentals of website development using HTML5 and CSS3. The website was created to practice semantic markup, page layout, navigation design, typography, and visual branding while simulating an online fashion boutique.

Rather than implementing e-commerce functionality, the project focuses on creating a polished static website with a consistent visual identity and intuitive page organization.
